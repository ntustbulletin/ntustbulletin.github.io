2021年"中技社科技獎學金&AI創意競賽"活動資訊

財團法人中技社為非營利公益法人，自創立以來一直以引進科技新知，培育科技人才，協助國內外經濟建設及增進我國生產事業之生產能力為目標，並設立獎學金資助培育人才長達數十年，以期對國家、社會及產業有所貢獻(https://www.ctci.org.tw/)。

本社2021年科技獎學金及AI創意競賽活動相關內容已於4月底函送至貴校，此附上相關資訊(下方說明)及附件，懇請您協助轉發活動訊息，期獎勵更多跨領域優秀人才。

2021年「中技社科技獎學金」

² 名額及獎額：

n   研究獎學金：15-30名，每名15萬元及獎狀

n   創意獎學金：15名(隊)；毎名(隊) 15萬元及獎狀

n   境外生研究獎學金：15-25名，每名15萬元及獎狀

n   境外生生活助學金：10-20名，每名10萬元，對象為經濟上有困難需資助者

² 申請期間：2021.07.01～2021.09.15

² 活動資訊：https://www.ctci.org.tw/8838/talent/ctci-scholarship/43618/43656/

2021年「中技社AI創意競賽」

² 競賽主題：AI與農林漁牧（智慧監控、精準生產、基因育種、自動採收、生態保育…等領域）

² 名額及獎額：前3名及佳作若干名

n   第一名：新台幣40萬元及獎狀

n   第二名：新台幣30萬元及獎狀

n   第三名：新台幣20萬元及獎狀

n   佳作：新台幣5萬元及獎狀

² 申請期間：2021.04.14～2021.09.10
